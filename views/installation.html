The compiler

C is a programming language that requires a compiler to be translated from readable language for human to readable code for the machine.

The are a lot of compilers but the most populars are:

  •  clang
  •  gcc

Ubuntu / Debian / Linux mint

$ sudo apt install gcc


Arch Linux

$ sudo pacman -S gcc


Gentoo

$ emerge gcc


Mac Os

$ brew install gcc
